## Ejecutar localmente

Clonar proyecto

```bash
  git clone https://github.com/kioshiokamoto/nextjs-portfolio-v1
```

Ir a ruta de proyecto

```bash
  cd nextjs-portfolio-v1
```

Instalar dependencias

```bash
  npm install
```

Iniciar cliente

```bash
  npm run dev
```
